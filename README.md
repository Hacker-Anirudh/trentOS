# trentOS
An open source web "OS" made by yours truly. You can learn more about me using it!

## Why?
Stardance, basically. And also it is pretty cool looking.

## Tech stack
HTML, CSS, and plain JS, no fancy TypeScript or JavaScript libraries here!

## Design
The design is inspired by the "old" (11-15) macOS design. However, it is deliberately poorly designed as to not infringe on any copyright, and because it's kind of a vibe.

Despite this, it's not really an OS in the traditional sense as you can do nothing with it, more like a fancy resume for myself.
